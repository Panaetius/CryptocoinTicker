CryptocoinTicker
================

A standalone ticker application for Cryptocoin exchanges.

Currently supports MtGox, BTCe, Bitstamp, Kraken and Vircurex.

Features:

- Scrollable/Zoomable Candle chart for different period sizes

- Market Depth

Prerequisites:

- .Net Framework 4.5
